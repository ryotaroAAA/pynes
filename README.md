## Installation
### Install poetry
- Mac
```
brew install poetry
```

- Other
```
curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python
```

### poetry install
```
poetry install --no-interaction
```

## (!) Usage
```
poetry run main
```
